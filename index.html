✓ Must Have: الدردشة الفورية، Push Notifications
✓ Should Have: الوضع الليلي
✗ Won't Have: دعم متعدد اللغات (مؤجل v3.1)
▸ المرحلة 2/5 — خريطة التبعيات (dependency-map)
⚠ تبعية حرجة: مراجعة App Store → 7-14 يوم
⚠ Backend يجب أن يسبق Mobile بأسبوع
▸ المرحلة 3/5 — سجل المخاطر (risk-register)
🔴 R1: تأخر App Store — احتمالية عالية / تأثير عالٍ [9]
🟡 R2: تعقيد WebSocket — احتمالية متوسطة [6]
▸ المرحلة 4/5 — Milestones
الأسبوع 1-2 → WebSocket Spike + Firebase
الأسبوع 3 → تطوير الدردشة + Push
الأسبوع 4 → Code Freeze + رفع App Store
الأسبوع 5 → UAT + Regression
الأسبوع 6 → إطلاق مرحلي 10%→50%→100%
▸ المرحلة 5/5 — قائمة التحقق قبل الإطلاق ✓

جاهز في دقيقتين

ثلاث طرق للتثبيت حسب الأداة التي تستخدمها.

# الطريقة الأسهل — بدون برمجة

1. افتح Customize (أسفل يسار)
2. اذهب لـ Browse plugins → Personal → +
3. اختر Add marketplace from GitHub
4. أدخل: devmuslim/pdm-skills

# جميع الـ 13 plugin تُثبَّت تلقائياً ✓
# Claude Code CLI

claude plugin add --marketplace devmuslim/pdm-skills

# ثم استخدم الأوامر مباشرة:
/plan-release وصف الإصدار هنا
/review-code الصق الكود هنا
/index-brd مسار مجلد BRD على Drive
/release-kpi اسم الإصدار في Jira
/assess-team الفريق كله
# نسخ المهارات لـ Gemini CLI أو Cursor أو Codex

for plugin in pdm-*/; do
cp -r "$plugin/skills/"* ~/.gemini/skills/ 2>/dev/null
done

# ملاحظة: الأوامر (/commands) تعمل فقط مع Claude
# المهارات (skills) تعمل مع أي AI يقرأ SKILL.md
