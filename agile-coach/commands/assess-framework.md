## الاستخدامات

### تقييم Framework محدد:
```
/assess-framework Ahmed — Kanban
/assess-framework Sara — Scrumban
/assess-framework الفريق — Kanban
```

### تقييم الالتزام بالمواعيد:
```
/assess-framework Ahmed — الالتزام
/assess-framework الفريق كله — الالتزام
```

### تقييم شامل:
```
/assess-framework Ahmed — كل شيء
```

## الخطوات

### إذا طُلب تقييم Framework:
1. اجلب أسئلة الـ Framework من skills/framework-assessment
2. اسأل 4-5 أسئلة مخصصة
3. قيّم على 4 محاور الـ Framework

### إذا طُلب تقييم الالتزام:
1. اسأل من س13 → س20 في skills/framework-assessment
2. إذا متوفر Jira → اجلب بيانات Sprint Completion Rate
3. أنتج تقرير الالتزام مع خطة تحسين

### إذا طُلب كل شيء:
- Framework Assessment: 5 أسئلة
- Delivery Commitment: 4 أسئلة
- أنتج تقرير موحد
