## أنتج خطة جلسة كاملة:

```
══════════════════════════════════
🎓 جلسة تعليمية: [الموضوع]
المدة: [X دقيقة] | المستوى: مبتدئ/متوسط/متقدم
══════════════════════════════════

🎯 هدف الجلسة
بعد هذه الجلسة سيكون الفريق قادراً على:
- [هدف 1]
- [هدف 2]
- [هدف 3]

📋 الأجندة
┌─────────────────────────────────┐
│ 00-05 | Warm-up / Ice breaker   │
│ 05-20 | الشرح النظري (موجز)    │
│ 20-40 | تطبيق عملي / لعبة      │
│ 40-55 | نقاش وأسئلة            │
│ 55-60 | Action item واحد       │
└─────────────────────────────────┘

🎮 النشاط التطبيقي
[وصف تفصيلي للنشاط التفاعلي]

❓ أسئلة للنقاش
1. [سؤال يثير التفكير]
2. [سؤال من تجربتهم]
3. [سؤال للتطبيق]

✅ Action Item
[شيء واحد يطبقه الفريق في Sprint القادم]

📚 للاستزادة
- [مورد مختصر — مقالة أو فيديو < 10 دقائق]
══════════════════════════════════
```
