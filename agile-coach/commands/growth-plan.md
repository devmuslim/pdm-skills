## الخطوة 0 — اجلب البيانات من المصادر الثلاثة

### من Jira:
```
- Velocity الشخصي مقارنةً بمتوسط الفريق
- Cycle Time مقارنةً بالهدف (< 5 أيام)
- Bug Rate هذا الشهر
- Stories مرتبطة بـ BRD vs غير مرتبطة
- Blocked Time (أيام blocked)
```

### من Google Drive:
```
- تقييم سابق في agile-coach/assessments/
- خطة تطوير سابقة في agile-coach/growth-plans/
  → إذا موجودة: قيّم التقدم أولاً قبل الخطة الجديدة
```

### من knowledge-base (إذا متاح):
```
- ما BRD's مرتبطة بمهامه؟
- هل هناك BRD's لم يقرأها بعد؟
```

## الخطوة 1 — حدد الفجوات بالأرقام

```
مثال:
الفجوة 1: Cycle Time = 8 أيام (الهدف 5) → فرق 3 أيام
الفجوة 2: Bug Rate = 15% (الهدف < 5%) → فرق 10%
الفجوة 3: BRD Linking = 40% (الهدف 100%) → فرق 60%
```
