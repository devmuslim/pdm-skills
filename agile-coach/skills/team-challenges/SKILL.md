## تحديات Sprint

### 🏆 تحدي الجودة
```
هدف: Sprint بدون Critical Bugs
المدة: Sprint كامل
الجائزة: لقب "Quality Champions" + نقاط للفريق كله
```

### ⚡ تحدي السرعة
```
هدف: إنهاء جميع Stories قبل آخر يوم في Sprint
المدة: Sprint كامل
الجائزة: يوم تعلم حر للفريق
```

### 🧠 تحدي المعرفة
```
هدف: كل عضو يشرح موضوعاً Agile للفريق
المدة: شهر
الجائزة: شهادة "Agile Champion"
```

### 🔍 تحدي المتطلبات
```
هدف: اكتشاف أكبر عدد من gaps في BRD قبل التطوير
المدة: لكل إصدار
الجائزة: نقاط مضاعفة + ذكر في تقرير الإصدار
```

## مسابقات شهرية

### 🥇 MVP of the Month
```
المعايير:
- أعلى نقاط في نظام النقاط
- مساهمة ملموسة في تحسين الفريق
- تصويت الفريق (50% من القرار)

الجائزة: شارة خاصة + عرض إنجازاته في اجتماع الشهر
```

### 💡 Best Improvement Idea
```
كل عضو يقترح تحسين في Retrospective
الفريق يصوت على أفضل فكرة
الفكرة تُطبَّق في Sprint القادم
```
