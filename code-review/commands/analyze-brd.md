## عند استلام وثيقة متطلبات:

1. حللها واستخرج تلقائياً:
   - Business Rules
   - Use Cases والتدفقات
   - الحقول والتحقق
   - الأخطاء والاستثناءات
   - الرسائل والإشعارات
   - التكاملات الخارجية

2. أنشئ Checklist منظمة جاهزة للاستخدام في:
   - `/review-code` — مراجعة الكود
   - `/review-figma` — مراجعة التصميم

3. اعرض ملخصاً:
```
### 📋 ملخص الوثيقة
- اسم المشروع: [X]
- عدد الـ Use Cases: [X]
- عدد Business Rules: [X]
- التكاملات: [X]

### ✅ Checklist جاهزة
[القائمة الكاملة]
```
