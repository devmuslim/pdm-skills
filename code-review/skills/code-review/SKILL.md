# مراجعة الكود — عام لأي مشروع

## متى تُفعَّل
عند مراجعة أي كود برمجي مقابل متطلبات أو BRD.

---

## منهجية المراجعة

### 1. إذا تم تحليل BRD مسبقاً:
استخدم الـ Checklist المستخرجة منه وتحقق من كل نقطة في الكود.

### 2. إذا لم يوجد BRD:
راجع الكود بناءً على:
- **الصحة الوظيفية** — هل الكود يحقق المطلوب؟
- **معالجة الأخطاء** — هل جميع الحالات مغطاة؟
- **الأمان** — هل توجد ثغرات واضحة؟
- **الأداء** — هل توجد مشاكل واضحة؟
- **جودة الكود** — قابلية القراءة والصيانة
