# Emotional Intelligence — قراءة الناس وإدارة الذات

## الـ 5 أبعاد (Goleman Model) — تطبيق عملي للـ Delivery Lead

---

## 1. Self-Awareness — اعرف نفسك تحت الضغط

```
سؤال يجب أن تسأله لنفسك كل يوم:
"ما المشاعر التي أحملها الآن إلى هذا الاجتماع؟"

علامات تحتاج توقف وتقييم:
- تفسّر كل اعتراض كهجوم شخصي
- تشعر بالدفاعية عند أي نقد
- تتسرع في القرارات تحت الضغط
- تتجنب أشخاصاً بعينهم

Triggers شائعة للـ Delivery Lead:
┌──────────────────────┬──────────────────────────┐
│ الـ Trigger          │ رد الفعل غير المدروس     │
├──────────────────────┼──────────────────────────┤
│ Scope يتوسع فجأة    │ الانسحاب أو التصعيد      │
│ فريق لا يلتزم       │ Micromanagement           │
│ Management يتدخل    │ الدفاع المفرط             │
│ صراع علني           │ الحسم السريع بدون استماع │
└──────────────────────┴──────────────────────────┘
```

---

## 2. Self-Regulation — إدارة نفسك في الموقف

```
تقنية STOP قبل أي رد صعب:
S — Stop:   توقف جسدياً (نَفَس عميق)
T — Think:  ما الذي يحدث فعلاً؟
O — Observe: ما مشاعري الآن؟ ما مشاعر الآخر؟
P — Proceed: ما الرد الذي يخدم الهدف؟

جمل تكسب وقتاً حقيقياً:
"سؤال مهم — خليني أفكر فيه لحظة"
"شاركني أكثر حتى أفهم بشكل صحيح"
"هل يمكنني أن أعود إليك بعد ساعة بإجابة أفضل؟"
```

---

## 3. Empathy — قراءة ما لا يُقال

```
ما يقوله الشخص ≠ ما يشعر به ≠ ما يحتاجه

نموذج الاستماع العميق:
┌─────────────────────────────────────────────┐
│ ما يقوله:   "هذه المهمة مستحيلة"           │
│ ما يشعر به: الإحباط + الشعور بعدم الكفاءة  │
│ ما يحتاجه: مساعدة في تقسيم المشكلة +       │
│            اعتراف بالصعوبة                 │
└─────────────────────────────────────────────┘

أسئلة تفتح الناس:
"ما الجزء الأصعب بالنسبة لك في هذا؟"
"لو كان الأمر بيدك، كيف كنت ستفعله؟"
"ماذا تحتاج مني حتى تنجح في هذا؟"
"ما الذي يقلقك أكثر؟"

إشارات غير لفظية للانتباه:
🔴 ذراعان متقاطعتان + صمت = مقاومة غير معلنة
🔴 الموافقة السريعة جداً = لم يقتنع
🔴 "لا مشكلة" بنبرة فاترة = توجد مشكلة
🟢 أسئلة تفصيلية = انخراط حقيقي
🟢 "وماذا لو..." = تفكير معك لا ضدك
```

## 4. Social Skills — بناء علاقات حقيقية

```
قاعدة 5:1
لكل تفاعل سلبي (نقد، رفض، ضغط)
يجب أن يسبقه 5 تفاعلات إيجابية حقيقية

ما يبني الثقة مع فريقك:
✅ الاعتراف بالخطأ علناً ("أخطأت في تقديري")
✅ تذكّر التفاصيل الشخصية ("كيف حال أخوك؟")
✅ الدفاع عنهم أمام Management
✅ إعطاء الفضل لهم علناً
✅ سؤالهم قبل القرار حتى لو قرارك نهائي

ما يهدم الثقة:
❌ تغيير الرأي تحت الضغط أمامهم
❌ الوعود التي لا تتحقق
❌ التذبذب بين أسلوبين حسب المزاج
❌ المعلومات المخفية "لحمايتهم"
```

---

## 5. تشخيص حالة الفريق العاطفية

```
مؤشرات Psychological Safety منخفضة:
- لا أحد يعترض في الاجتماعات
- الاعتراضات تأتي بعد الاجتماع (Hallway conversations)
- الأخطاء تُخفى حتى تتفاقم
- "نعم" الجماعية على كل قرار

مؤشرات Burnout خفي:
- انخفاض جودة الكود فجأة
- تأخر في الرد على الرسائل
- غياب المبادرة
- "أنهيت المهمة" بدون اهتمام بالجودة

مؤشرات صراع مكبوت:
- اجتماعات هادئة جداً مع خلافات خارجها
- تكتلات (من يجلس مع من)
- تبادل الاتهامات غير المباشرة في Code Review
- "هذا مش شغلتي" يتكرر
```

## أداة التقييم العاطفي السريع — قبل كل 1:1

```
اسأل نفسك قبل الاجتماع:
1. ما آخر مرة قدّرت هذا الشخص علناً؟
2. هل أنا أعرف ما الذي يقلقه هذا الأسبوع؟
3. هل هو يعرف أنني أثق به؟
4. ما الذي ينتظر أن يسمعه مني اليوم؟
5. ما الذي أتجنب قوله له؟ لماذا؟
```
