# Command: /handle-conflict
## Usage: `/handle-conflict [صف الصراع: من؟ بين من؟ متى بدأ؟ ما الذي حدث؟]`
## Workflow: conflict-resolution skill → تشخيص النوع → خطة تدخل مرحلية → أجندة جلسة

## Description
يشخّص نوع الصراع ويبني خطة تدخل مرحلية مع أجندة جلسة جاهزة وجمل تقولها.

## Steps
1. اجلب skills/conflict-resolution/SKILL.md
2. اجلب skills/situational-playbook/SKILL.md
3. شخّص النوع: مهام / عملية / علاقات / قيم
4. حدد مرحلة الصراع: خفي / ظاهر / متفاقم
5. قرر: هل تتدخل الآن أم تعطي مساحة؟
6. ابنِ خطة مرحلية (1:1 ثم مشترك)
7. أنتج أجندة الجلسة المشتركة بالوقت
8. أعطِ جملاً جاهزة للمواقف الصعبة في الجلسة

## Output Format
```
⚖️ خطة إدارة الصراع
━━━━━━━━━━━━━━━━━━━━

🔎 التشخيص:
النوع: [مهام / عملية / علاقات / قيم]
المرحلة: [خفي / ظاهر / متفاقم]
الجذر المحتمل: ...

📋 خطة التدخل:
[ ] اليوم: 1:1 مع [الطرف الأول]
[ ] غداً: 1:1 مع [الطرف الثاني]
[ ] بعد يومين: الجلسة المشتركة (إذا كانا مستعدَّين)

🗓️ أجندة الجلسة المشتركة:
[5 دق] القواعد: ...
[15 دق] الطرف الأول: ...
[15 دق] الطرف الثاني: ...
[10 دق] الأرضية المشتركة: ...
[15 دق] الاتفاق: ...

💬 جمل جاهزة:
لإيقاف الهجوم: "..."
لإعادة التركيز: "..."
للإغلاق: "..."
```
