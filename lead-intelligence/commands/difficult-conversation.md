# Command: /difficult-conversation
## Usage: `/difficult-conversation [من؟ ما الموضوع؟ ما الهدف من المحادثة؟]`
## Workflow: communication skill → SBI/COIN → صياغة كاملة → أسئلة متوقعة + ردود

## Description
يصيغ محادثة صعبة كاملة — الافتتاح، الجوهر، الإغلاق — مع ردود على الاعتراضات المتوقعة.

## Steps
1. اجلب skills/communication/SKILL.md
2. اجلب skills/emotional-intelligence/SKILL.md
3. طبّق SBI + Intent أو COIN حسب الموقف
4. اكتب المحادثة كاملة:
   - الافتتاح (بناء الأمان النفسي)
   - الجوهر (SBI/COIN)
   - الاستماع (أسئلة مفتوحة)
   - الاتفاق (N في COIN)
5. توقع 3 ردود صعبة محتملة + كيف تتعامل معها
6. أعطِ نصائح للـ Tone والـ Body Language

## Output Format
```
🗣️ محادثة: [الموضوع]
الهدف: [ما تريد تحقيقه]
━━━━━━━━━━━━━━━━━━━━━━

🚪 الافتتاح (بناء الأمان):
"..."

💬 الجوهر (SBI/COIN):
S/C: "..."

N:   "..."

- "..."
- "..."

"..."

رد 3: "..." → ردك: "..."

- Tone: ...
- التوقيت المناسب: ...
- المكان المناسب: ...
```
