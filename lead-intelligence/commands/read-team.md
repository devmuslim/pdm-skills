# Command: /read-team
## Usage: `/read-team [صف ما تلاحظه في الفريق مؤخراً]`
## Workflow: emotional-intelligence skill → تشخيص الحالة → خطة تدخل فردية وجماعية

## Description
يحلل الإشارات التي تصفها ويشخص الحالة العاطفية للفريق — Burnout؟ صراع مكبوت؟ Safety منخفضة؟

## Steps
1. اجلب skills/emotional-intelligence/SKILL.md
2. اجلب skills/situational-playbook/SKILL.md
3. حلل الإشارات:
   - ما الذي يقوله الناس؟
   - ما الذي لا يقولونه؟
   - ما الأنماط السلوكية الملاحظة؟
4. شخّص الحالة: Burnout / صراع مكبوت / Safety منخفضة / غياب الدافعية
5. ابنِ خطة تدخل:
   - على المستوى الفردي (من يحتاج 1:1 فوري؟)
   - على مستوى الفريق (ما الذي يجب تغييره هيكلياً؟)
6. أعطِ أسئلة تطرحها في الـ 1:1 القادم

## Output Format
```
❤️ تشخيص الحالة العاطفية للفريق
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

🔍 الإشارات التي لاحظتها:
...

🧠 التشخيص:
الحالة الرئيسية: [Burnout / صراع / Safety / دافعية]
المستوى: [فردي / جماعي]
الخطورة: [🟢 مبكر / 🟡 متوسط / 🔴 عاجل]

📋 خطة التدخل:
فوري (هذا الأسبوع):
- ...

متوسط المدى (هذا الشهر):
- ...

💬 أسئلة للـ 1:1 القادم:
- "..."
- "..."
- "..."

⚠️ ما تتجنبه:
- ...
```

## Integration
- إذا كشف صراعاً → يقترح /handle-conflict
- إذا احتاج محادثة صعبة → يقترح /difficult-conversation
- إذا طلب بيانات Jira → يتكامل مع agile-coach /team-progress
