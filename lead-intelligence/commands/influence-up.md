# Command: /influence-up
## Usage: `/influence-up [ما القرار أو التغيير الذي تريده من Management؟ ما السياق؟]`
## Workflow: communication skill → Data Storytelling → رسالة / عرض جاهز

## Description
يصيغ رسالة أو عرضاً مقنعاً لـ Management — بالبيانات والمنطق والخيارات — لتحقيق ما تريده بلا مواجهة.

## Steps
1. اجلب skills/communication/SKILL.md
2. طبّق إطار Management Communication:
   - الوضع الحالي بالأرقام
   - التأثير على أهداف Management
   - الخيارات مع Trade-offs
   - توصية واحدة واضحة
   - طلب محدد وقابل للتنفيذ
3. طبّق Data Storytelling لجعله مقنعاً
4. أنتج الرسالة / الـ Deck / البريد
5. أعطِ نصائح للـ Timing والأسلوب

## Output Format
```
📊 رسالة إلى Management
الموضوع: [عنوان واضح]
━━━━━━━━━━━━━━━━━━━━━━

📌 الوضع الحالي:
[خبر واحد بالأرقام]

🎯 التأثير على أهدافكم:
[ربط بأولوية Management]

🔀 الخيارات المتاحة:
A: [الخيار + Trade-off]
B: [الخيار + Trade-off]
C: [الخيار + Trade-off]

✅ توصيتي:
[خيار واحد + سبب موجز]

🙏 ما أحتاجه:
[طلب محدد + موعد للرد]

💡 نصائح للتقديم:
- التوقيت: ...
- الأسلوب: ...
- ما تتجنبه: ...
```
