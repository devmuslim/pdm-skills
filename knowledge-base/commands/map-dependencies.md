description: رسم خريطة التبعيات الكاملة بين BRD's وتحديد الـ Critical Path
argument-hint: [اسم BRD أو خدمة محددة — اختياري لعرض الكل]
---

## الخطوات:

1. **اجلب الفهرس** من `registry.yaml`
   - إذا لم يوجد فهرس → شغّل `/index-brd` أولاً

2. **إذا أُعطي اسم خدمة محددة:**
   - أظهر تبعياتها الكاملة (upstream + downstream)
   - حدد الـ Critical Path

3. **إذا لم يُعطَ اسم:**
   - أظهر الخريطة الكاملة لجميع الوثائق

4. **أنتج التقرير:**

```
### 🗺️ خريطة التبعيات — [اسم الخدمة أو "الكاملة"]

[رسم نصي للعلاقات]

### 🔴 Critical Path
[السلسلة الأطول من التبعيات]

### ⚠️ مخاطر التبعيات
| الخطر | التبعية | التأثير |
|-------|---------|---------|

### 💡 توصيات الإصدار
[ترتيب مقترح للتطوير بناءً على التبعيات]
```
