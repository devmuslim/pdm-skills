# Command: /map-dependencies

## Description
Map all delivery dependencies for a release — teams, systems, external parties — and flag the critical path.

## Usage
```
/map-dependencies [describe teams and systems involved]
```

## Workflow
1. Run `dependency-map` skill
2. Run `risk-register` skill for dependency-driven risks
3. Produce dependency table + critical path + risk flags

## After Completion
→ `/plan-release` or `/risk-register`
